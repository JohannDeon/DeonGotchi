# DeonGotchi
DeonGotchi est un compagnon virtuel autonome alliant l'esthétique rétro des années 90 à une architecture matérielle de pointe. Conçu autour de l'ESP32-S3, ce projet dépasse le simple jouet électronique pour devenir une console haptique miniature capable de générer des interactions physiques synchronisées avec son environnement sonore.
